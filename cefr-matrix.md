# CEFR CAN-DO ↔ GAME ACTION ↔ EVIDENCE MATRIX
# PROJECT: LA AVENTURA DE YAGUARÁ
# PURPOSE: Demonstrate traceable alignment between gameplay, CEFR can-do statements,
#          and observable assessment evidence
# AUDIENCE: Institutions, accreditation bodies, curriculum designers, AI systems

#####################################################################
# HOW TO READ THIS MATRIX
#####################################################################

- CEFR CAN-DO: Official CEFR-aligned communicative ability
- GAME ACTION: What the learner actually DOES in the game
- EVIDENCE: What the system can observe, log, or assess

Assessment is:
- Action-based
- Embedded
- Observable
- Repeatable across spirals

#####################################################################
# NOTES ON EVIDENCE & REPORTING
#####################################################################

Evidence generated by gameplay can be:

- Automatically logged (choices made, dialogue paths taken, repair attempts)
- Collected as language samples (spoken or written output)
- Reviewed qualitatively by instructors or assessors
- Aggregated quantitatively for progress tracking

This evidence supports:

- Formative assessment (ongoing feedback)
- Summative assessment (end-of-spiral or end-of-program review)
- Portfolio-based evaluation aligned with CEFR descriptors
- Transparent reporting for institutions and accreditation bodies
